# 游戏
名词/动词

游戏，既可以指人的一种娱乐活动，也可以指这种活动过程。游戏的道具可以为玩具。在英语，体育比赛（Game）亦是游戏的一种，而体育运动亦是由游戏演变出来。游戏是一种有组织的玩耍，一般是以娱乐为目的，有时也有教育目的[1]。游戏不同于会有对应金钱报酬的工作，也不同于呈现美学或是概念元素的艺术。不过彼此之间的分界不一定很明确，像职业运动员的游戏和工作可能是一体，而像拼图游戏则同时具有游戏和艺术的成分在内。

游戏的主要成分有目的、规则、挑战及互动。游戏一般会有心理或是身体上的刺激，许多游戏可以培养相关技巧，有体能性、教育性、模拟性或心理上的意义。

在生活场景中，游戏往往指代电子游戏，包括使用电脑、手机、平板、主机等电子工具进行的游戏，而乒乓球、网球、篮球、斯诺克更多被认为是体育运动，围棋、象棋等被认为是智力竞技。但这些都可以认为是广义上的游戏。

# 游戏为什么会让人沉迷
及时反馈机制，努力就能得到回报。世界上可能没有其他的东西类似于游戏这样，可以让你努力就能获得回报：在MMORPG中，经验值是一个可视化的进度条，每一次杀怪，就可以看到经验值噌噌噌的往上涨；在dota中，你补到的每一刀实实在在转化成了金币进而转化成了身上的装备，等蝴蝶bkb撒旦一出，带领着队友就把对面的基地给掀了；在csgo中，充值不能使你变强，但是跑图和练枪可以，日积月累增加的游戏经验会让你变得越来越像一名出色的反恐精英/恐怖分子。

反之，现实生活往往那么的令人沮丧。永远匮乏的金钱，不断被挤压的个人时间和空间，永远听不懂你说话的家人，难缠的客户，弱智的领导，连带着根本不可能完成的业绩任务，为什么一定要在地球OL这个垃圾游戏中摸爬滚打呢，沉迷在游戏世界中当令人（NPC)崇拜的救世主不好吗？

# 为什么老一辈的家长认为游戏是洪水猛兽
我认为有两个原因：

第一个原因是老一辈的家长没有认识到游戏的价值，游戏其实是一个很好的能力学习工具以及沉浸式的模拟器。就我个人来说，是csgo教会了我团队沟通和经济互助，是dota教会了我自私和自信，求人不如求己，队友被抓边笑边刷，是暗黑地牢教会了我面对机会和挑战时要做好充分的准备，是艾尔登法环教会了我要克服恐惧的最好办法就是面对恐惧。我那些游戏玩的好的同学，往往是在学生时代是最聪明的那一批，后来很多也在职场和生活中顺风顺水，而那些成绩差的孩子，很多连游戏也玩不好。好的游戏数值设计的都非常合理，如何运用有限的游戏资源，去解决游戏给出的各种难题，是一个很好的可以锻炼问题解决能力的方式。

另一个原因就是老一辈家长认为自己的权威受到了挑战——“我跟你都说了不要玩游戏，你竟然敢不听？”，结果就是这样的家长自己也管理不好自己，整天抱着手机刷抖音，俨然成为了多巴胺的奴隶。这类家长老搞严于律人，宽于待己这一套，孩子们自然也不服气，但孩子一旦尝试辩解，就会认定为“不听话”，亲子关系随之就会出现问题。

## 教育失败不能归咎于游戏
这类问题的极端体现就是，家长们把这类“网瘾少年”送入了网瘾治疗学校，如雷电法王杨永信，“豫章书院”等各类网瘾治疗学校，这些学校的真正目的是哪些家长手中的钱，而用军事化的方式折磨孩子，最后孩子成为了木头人，留下了一生都挥之不去的阴影，家长却以为孩子懂得了“感恩”，以为孩子被治好了。

# 怎样才能“不玩游戏”。
如果你问AI，或者在小红书抖音上面看到一些“成功”的家长，他们会跟孩子约定，比如每天只能玩一个小时，或者是每周只能玩2个小时的游戏，我就是在这样的环境成长起来的，我成长的年代电子游戏刚刚兴起，我妈就规定我每周回来只能玩2个小时，导致我非常渴望玩游戏，上学上课放学，甚至是旅游的时候无时不想着游戏里的场景。我跟艾欣爸爸讨论过，如果柚子以后沉迷游戏怎么办，他的思路就是这个，他表示艾欣现在就不喜欢打游戏，那我能说什么呢？我只能说他的观点“基本正确”（当别人说你的观点或言语基本正确的时候，往往代表着你错的离谱），实际情况是艾欣非常喜欢打游戏，是饥荒、缺氧、星露谷物语的骨灰级玩家，截至目前我们已经结婚2年多了，她基本每天晚上睡觉前都会打游戏，打到很晚很晚，而我通常都扛不住先睡了。

反观我自己，我是dota、cs的骨灰级老玩家，其他主机、单机游戏的爱好者，直到今天，我才慢慢的觉得游戏不好玩了，我买了switch、PS，Steam库存里超过200个游戏，dota2综合玩了近6000小时，cs近乎1500小时，但是今天，我已经失去了打开他们的动力。你问我为什么？我会觉得游戏也没有那么好玩了，dota2往往化身为坐牢模拟器，cs如果不练枪，枪法就会马的自己都看不下去，其他的单机游戏，很难带给我心流的体验，至于“免费”的手游、MMORPG类型的网游，我就从来没有在这种类型的游戏里面找到过快乐，更别说免费的游戏往往最贵了。而我找了一些更能让我进入“心流”状态的事物，比如斯诺克，比如麻将，比如阅读。

所以，怎么才能让孩子“不玩游戏”呢？我的答案就是让孩子玩市面上最好的游戏，游戏审美提高后，哪些画质垃圾，玩法垃圾的游戏自然吸引不了孩子；由这个思路发散一下，有两个推论，一是只有当一个人的审美提高之后，他才不会沉迷世俗的娱乐活动；二是人生需要体验各式各样的事物，去魅之后，自然就不会被这些魑魅魍魉所迷惑。

怎么才能让孩子不被绿茶渣男骗呢？答案就是鼓励孩子多和异性接触，更早的理解为什么人与人之间最稳定的关系是“对等”。

怎么才能让孩子不沾上“赌瘾”呢？答案就是让孩子自己体验一下，把所有的钱输光之后就不会赌博了。

但是对自身身体会造成伤害的，比如“毒”、会违反法律和公序良俗的行为，就是不应该去做的。

一件事情，如果不能给人带来快乐，很难坚持下去。豫章书院和杨永信折磨出来的学生，对游戏失去了兴趣，也对生活、生命失去了兴趣。同样的，这个视角来看，兴趣、快乐是最好的老师，对孩子来说一定要保护好好奇心，对世间万物都感到兴奋与好奇，因为这是生命力的表现。

# 有心无肺，游戏人生
世界上真的和游戏不一样，努力不一定能获得回报吗？其实不然，世界上的所有东西都是人们通过努力得来的。

人生其实可以把他看成是一个大型全拟真MMORPG《地球Online》，在出生条件限定的情况下，你可以一定程度上选择如何度过你的一生，只不过这个游戏没有游戏界面，你唯一可以确认的是不断增长的年龄，不断成长然后衰老的身体，你的银行账户余额及固定资产（如果私有财产不能得到保护的话， 连这个也不能确定）。

而更加关键的，人生重大路口走向抉择、人物对话框选项、人际关系好感度、职业技能和生活技能兴趣爱好等能力的经验条、主线及支线任务触发方式以及完成进度等等游戏要素全部都是隐藏的而在这个游戏中，如果你没有足够的人生阅历（经验），你根本无法意识到他们的存在，更不用说要处理好这些事情了。会产生恐惧、不自信都是很正常的，而遇到困难最好的办法就是克服恐惧的面对困难；自信来源于自身的能力，自身的能力又是通过不断的练习、学习、实践得来的。

尽管当下的中国还存在着种种问题（比如2025年，经济发展减缓，失业率居高不下，工作强度与压力巨大，社会分配不公，国家言论管制趋紧，文化审核苛刻，法治路途遥远，践踏私有产权等等），但这已经是自虞夏商周以来的中国，生产力水平最高，人均财富最多，经济发展最好，治安情况最好的时代了。

在从农耕文明迈入工业文明的中国里，人们可以选择从事的职业变多了，可以发展的兴趣爱好也变多了。当你有一天可以找到自己喜欢的事物，并为之奉献出全部的激情和热血的时候，也就是你的生命真正的绽放出光彩的时候；而作为一个“普通人”，普通的度过这一生也无不可，相比于世俗意义的成功，作为父辈的我们更希望你们可以健康、正直、独立、幸福。
